# Credit-Card-Analysis – Power BI Project

Project Overview

This project presents a Power BI dashboard that provides real-time insights into key performance metrics of a credit card business. The goal is to help stakeholders monitor revenue, customer activity, and card performance to support data-driven decisions.

Objectives

Analyze weekly revenue trends and transaction performance.

Understand customer demographics and income segments.

Track card activation rate and delinquency rate.

Identify top-performing card types and regions.

Datasets

credit_card.csv – Contains financial and transaction details.

Columns: Client_Num, Card_Category, Annual_Fees, Activation_30_Days, Customer_Acq_Cost, Week_Start_Date, Total_Trans_Amt, Interest_Earned, Delinquent_Acc, etc.

customer.csv – Contains demographic and personal details.

Columns: Client_Num, Customer_Age, Gender, Education_Level, Income, Marital_Status, Cust_Satisfaction_Score, etc.

Both datasets are joined using the Client_Num column for unified analysis.

Tools & Technologies

Power BI – Dashboard creation and visualization

Excel / CSV – Data source format

DAX – Custom calculations and measures

Key Insights

Total Revenue: $57M | Interest Earned: $8M | Transaction Amount: $46M

Activation Rate: 57.5% | Delinquency Rate: 6.06%

Blue & Silver cards generate over 90% of total revenue.

TX, NY, and CA are the top revenue-generating states.

Majority of customers are educated, married, and aged 30–60.

Recommendations

Improve activation through onboarding rewards.

Reduce delinquency with reminders and flexible payments.

Expand marketing in underperforming regions.

Introduce premium or cashback card variants.

Use Power BI for continuous performance monitoring.

Project Deliverables

Power BI Dashboard (.pbix file)

Processed datasets (credit_card.csv, customer.csv)

Project report presentation (.pptx)
